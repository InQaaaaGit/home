# CDO Showcase Tools

Административный плагин Moodle для демонстрации функциональности и возможностей CDO.

## Особенности

- Полная совместимость с Moodle 4.5+
- Соответствие современным стандартам PHP 8.1+
- Следование принципам SOLID, KISS, DRY
- Поддержка Privacy API
- Система прав доступа
- Модульная архитектура

## Структура плагина

```
├── version.php              # Метаданные плагина
├── settings.php             # Настройки администрирования
├── index.php                # Главная страница плагина
├── lib.php                  # Основные функции плагина
├── lang/en/                 # Языковые файлы
│   └── tool_cdo_showcase_tools.php
├── db/                      # Схема базы данных
│   └── access.php           # Определение прав доступа
└── classes/                 # Классы плагина
    ├── manager.php          # Основной класс управления
    └── privacy/             # Privacy API
        └── provider.php
```

## Права доступа

- `tool/cdo_showcase_tools:view` - Просмотр инструментов
- `tool/cdo_showcase_tools:manage` - Управление инструментами

## Установка

1. Скопируйте папку плагина в `admin/tool/cdo_showcase_tools/`
2. Войдите в систему как администратор
3. Выполните обновление плагинов через интерфейс администратора

## Использование

После установки плагин будет доступен в разделе:
**Администрирование сайта → Плагины → Административные инструменты → CDO Showcase Tools**

## Архитектура

Плагин построен с использованием современных подходов:

- **Singleton Pattern** для менеджера
- **Dependency Injection** готов к использованию
- **Privacy API** полностью реализован
- **Namespace** структура для автозагрузки классов

## Разработка

Для расширения функциональности:

1. Добавьте новые методы в класс `manager`
2. Создайте соответствующие языковые строки
3. Обновите права доступа при необходимости
4. Следуйте принципам SOLID при добавлении новых классов

## Совместимость

- **Moodle**: 4.5+
- **PHP**: 8.1+
- **Базы данных**: MySQL 8.0+, PostgreSQL 13+
